How to run this Project

1. Download and Unzip file on your local system copy library.
2. Put library folder inside root directory
3. Please run the project using xampp server (phpMyAdmin)
4. Paste the folder [airline-ticket-reservation] inside [xampp\htdocs] folder.

-------------------

Database Configuration

Open phpmyadmin (xampp local server [localhost])
Create Database airline_reservation
Import database airline_reservation.sql (available inside zip package) [FastAirline\airline-ticket-reservation\Database file\airline_reservation.sql]

-------------------
Go to browser -> copy and paste this link into the address bar [ localhost:8080/airline-ticket-reservation/home_page.php ]
